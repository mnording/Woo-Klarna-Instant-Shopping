Adding Klarna instant shopping functioality to Wordpress

== Description ==
Allowing you to display Klarna Instant Shopping buttons on Simple and Variable products. 
https://www.klarna.com/uk/business/products/instant-shopping/

== Installation ==
Upload to wp-plugins
install from dashboard

== Frequently Asked Questions ==
*What is Instant Shopping?*

Read more at the Klarna Product page - https://www.klarna.com/uk/business/products/instant-shopping/

*What product types does this plugin support?*
This plugin will activate the button on Simple and Variable products
